Create customizable lightbox with any media or content like images, videos and slideshows.

= Features =

* Responsive
* Different lightbox effects
* Multisite support

== Installation ==

= To install a plugin via FTP =

1. Download the plugin
2. Unarchive the plugin
3. Copy the folder with plugin
4. Open ftp \wp-content\plugins\
5. Paste the plug-ins folder in the folder
6. Go to admin panel => open item "Plugins" => activate the plugin
